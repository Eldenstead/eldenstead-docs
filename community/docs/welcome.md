# Community Docs

Eldenstead is a medieval fantasy roleplay survival server with kingdoms, empires, custom weapons, mounted combat, skill progression, economy systems, NPCs, rituals, and lore-driven progression.

Use these docs to understand how the public server systems work before asking staff for help.

## Core Areas

- Start with joining steps and roleplay expectations.
- Create and maintain a character card.
- Learn how bloodlines, messenger birds, kingdoms, skills, and combat systems fit together.
- Use the forum and Discord for announcements, support, and community coordination.
