# Character Cards

Character cards define the identity other players and plugins use for roleplay systems.

## What a Card Contains

- Character name
- Description
- Age
- Gender
- Race
- Optional parent links

## Useful Commands

```text
/character
/character create <name> <description...> <gender> <race> <age>
/character card [player]
/character set name <value>
/character set description <value>
```

Complete character cards are required before using bloodline and marriage features.
