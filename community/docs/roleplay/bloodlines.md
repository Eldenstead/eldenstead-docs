# Bloodlines

Bloodlines represent family or lineage groups for roleplay identity and inheritance.

## Core Concepts

- Bloodlines have leaders, elders, and members.
- Leaders can invite members, change ranks, set crests, and delete the bloodline.
- Elders can invite members and set crests.
- Family protection blocks close-family relationships that would break continuity rules.

## Useful Commands

```text
/bloodline
/bloodline create <name>
/bloodline invite <player>
/bloodline accept
/bloodline info [name]
```
